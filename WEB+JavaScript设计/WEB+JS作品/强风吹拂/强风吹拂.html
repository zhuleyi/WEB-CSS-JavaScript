书籍简介:

三浦紫苑的同名小说《强风吹拂》，有如沧海遗珠。这部关于长跑的半年番，开播之初就口碑爆棚。直至今天，它依旧保持着极为优异的口碑，豆瓣评分高达9.6，以及B站超过8万人打出的9.9的逆天分数。作为三浦紫苑第二部动画化的作品，《强风吹拂》有着与之前《编舟记》相似的基因和境遇。它们都烙印着作家鲜明的个人风格，在平缓冷静的叙述中流淌出涤荡人心的力量，内里传达的对于某个目标始终如一的坚持也如出一辙。而且，它们都是有口皆碑的冷门佳作。
